# 2023 Personal Project 
## Autonomous Robot Navigation
Project Goals: 
- Navigate Robot Autonomously
  - Learn ROS2 (Robot Operating System) fundamentals
  - Learn NAV2 navigation stack features
  - Learn SLAM (Simultaneous Localization and Mapping) toolbox
  - Learn how to develop / test robot in Gazebo simulation environment

Stretch goals:
- Navigate robot autonomously in real world
